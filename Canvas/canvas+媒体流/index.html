媒体流+canvas应用：自定义卡片拍照

问题背景： 由于h5通过 <input type="file" /> 方式吊起拍照的为系统相机，给用户的体验并不是很好，没有裁切框，也无法在系统相机上附加 tips 蒙层进行扩展，比如在蒙层上告知用户拍照的注意事项。所以业务上需要实现一个自定义拍照卡片的页面。

本文重点 API：MediaDevices.getUserMedia()

1、前期准备工作-兼容性问题​

本文使用的 api 会有兼容性问题，如果遇到用户不提供摄像头权限，或者出现以下的任意一个错误，请考虑兼容方案（调用系统相机）

  • AbortError［中止错误］
  • NotAllowedError［拒绝错误］
  • NotFoundError［找不到错误］
  • NotReadableError［无法读取错误］
  • OverConstrainedError［无法满足要求错误］
  • SecurityError［安全错误］
  • TypeError［类型错误］

2、MediaDevices.getUserMedia()​

MediaDevices.getUserMedia() 会提示用户给予使用媒体输入的许可，媒体输入会产生一个MediaStream，里面包含了请求的媒体类型的轨道。此流可以包含一个视频轨道（来自硬件或者虚拟视频源，比如相机、视频采集设备和屏幕共享服务等等）、一个音频轨道（同样来自硬件或虚拟音频源，比如麦克风、A/D 转换器等等），也可能是其他轨道类型。

它返回一个 Promise 对象，成功后会resolve回调一个 MediaStream 对象。若用户拒绝了使用权限，或者需要的媒体源不可用，promise会reject回调一个 PermissionDeniedError 或者 NotFoundError 。

3、能力检测​

由于不同浏览器对于标准的实现不一致，需要作api能力的兼容，避免用户浏览器无法正常调用该api。

function getUserMedia(constrains) {
  if (navigator.mediaDevices?.getUserMedia) {
    //最新标准API
    return navigator.mediaDevices.getUserMedia(constrains);
  } else if (navigator.webkitGetUserMedia) {
    //webkit内核浏览器
    return navigator.webkitGetUserMedia(constrains);
  } else if (navigator.mozGetUserMedia) {
    //Firefox浏览器
    return navigator.mozGetUserMedia(constrains);
  } else if (navigator.getUserMedia) {
    //旧版API
    return navigator.getUserMedia(constrains);
  }
  return Promise.reject('访问用户媒体设备失败：')
}

4、调用 api 的前置元素：video​

需要在页面上放置 video 组件，根据自己的需求调整样式。

<video

autoplay-自动播放
muted-无声音

5、获取用户媒体流​

// 调用媒体设备，访问摄像头
function getUserMediaStream(videoNode) {
  // 调用用户媒体设备，访问摄像头
  // 上面的方法，兼容获取浏览器的 Media
  getUserMedia({
    audio: false,
    // video: { facingMode: { exact: 'environment' } },
    // video: true,
    video: {
      facingMode: { exact: 'environment' },
      width: { ideal: 1280 },
      height: { ideal: 720 },
    },
  }).then(res => {
    mediaStream.value=res;
    return handleSuccess(res, videoNode);
  })
  .catch(error => {
    console.log('访问用户媒体设备失败：', error.name, error.message);
    return Promise.reject();
  });
}

// 调用访问成功的回调
function handleSuccess(stream, video) {
  return new Promise((resolve, reject) => {
    video.srcObject = stream;
    //播放视频
    video.onloadedmetadata = function (e) {
      video.play();
      resolve(true);
    };
  });
}

7、点击拍照后根据页面进行裁切​

拍照后使用canvas进行裁切图片。

// 点击拍照事件，处理canvas生成图片
function handleTakePhoto() {
  const video = videoRef.value;
  const canvas = canvasRef.value;
  if(canvas&&video) {
    const context = canvas.getContext('2d');
    context?.clearRect(0, 0, canvas.width, canvas.height);
    canvas.width = calcWidth.value;
    canvas.height = calcHeight.value;
    const videoRatio = video.videoHeight/video.videoWidth;
    const truthWidth = video.offsetHeight/videoRatio;
    const offsetLeft = (truthWidth-video.offsetWidth)/2
    // 具体数字要根据自己的布局去计算
    context?.drawImage(video, -offsetLeft-20, -42, truthWidth, video.offsetHeight);
    const photoURL = canvas.toDataURL('image/jpeg');
    // 在这里可以对拍摄的照片进行进一步的处理或显示
    // console.log('拍照成功！照片URL:', photoURL);
    imgsrc.value=photoURL;
  }
};
