使用 Vue3+Vite 在 VScode 中开发 uni-app

问题背景： 跨端项目需要使用 uni-app 进行开发，uni-app 有自带依赖的编辑器 HBuilder，因为个人开发习惯使用 Vscode，所以研究了如何在 vscode 中更好的开发 uni-app，uni-app 也支持 Vue3+Vite 进行开发，这里一并总结一下经验。

1、初始化项目​

HBuilder 自带依赖，所以如果在 Vscode 中开发，则需要下载对应依赖，访问这里下载Vue3+Vite开发模板。

项目安装好之后下载依赖

yarn

运行

yarn dev:xxx

2、安装样式处理器​

安装 node-sass，sass；

npm i -D sass
npm i -D node-sass

3、配置 Eslint​

  • 命令：npm i -D eslint;
  • 初始化 eslint：npx eslint --init;

4、运行小程序​

Vscode 不会自动在微信开发者工具中导入项目，启动小程序需要下面两步：

  • 1、先 yarn dev:mp-weixin，在 dist/dev/mp-weixin 生成小程序文件
  • 2、手动打开微信开发者工具并导入项目，PS：仅首次启动需要导入

5、增强 pages.json 和 manifest.json 开发体验​

我们打开pages.json和manifest.json，发现会报红，这是因为在json中是不能写注释的，而在jsonc是可以写注释的。

解决方案：我们把 pages.json 和 manifest.json 这两个文件关联到jsonc中，然后就以写注释了。在设置中打开settings.json，添加：

"files.associations": {
  "pages.json": "jsonc",
  "manifest.json": "jsonc"
},

注: json 就是 json，jsonc 就是 jsonc，这两个是不一样的，如果在 package.json 写注释 VSCode 不报错了，但编译的时候还是会报错的。

6、增强插件​

  • uni-app-schemas：提供语法提示和校验，使开发更加方便
  • image preview：鼠标悬停可以预览图片
  • uni-create-view：支持一键创建
  • volar(Vue Language Features): 为 Vue3 提供的语法扩展插件

  • @dcloudio/types，uni语法提示
  • wechat-miniprogram，微信语法提示
