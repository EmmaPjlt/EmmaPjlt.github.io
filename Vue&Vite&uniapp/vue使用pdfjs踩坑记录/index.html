vue 使用 pdfjs 踩坑记录

1、安装 pdfjs​

yarn add pdfjs-dist

2、踩坑报错解决​

2.1 Cannot read from private field​

引入后控制台报错，报错内容：Cannot read from private field 读不到私有变量。

经排查是 pdfDoc.getPage 出了问题,和 vue3 的 proxy 有关

  • vue2 是通过 defineProperty 来监听数据的 set 和 get;
  • vue3 是通过 proxy 来监听数据的。

在 pdfjs-dist 的源码中，做了一个拦截校验，校验内容就是当前传入的参数，是否有 obj 对象，如果没有的话，直接抛出读不到私有变量错误。在之前老项目中，这样写是没有问题的。在 vue3 中， pdfjs-dist 拦截校验的时候，获取到的是一个 proxy 对象，所以会直接抛出异常。

正确解法：pdfDoc.getPage 的 pdfDoc 变量不要使用响应式写法(ref、reactive等)

2.2、引入 PdfWorker 编译报错​

import * as PDFJS from 'pdfjs-dist/legacy/build/pdf.js'
// 这里导入的是 pdf.worker.entry.js
import * as pdfjsWorker from 'pdfjs-dist/legacy/build/pdf.worker.entry.js'

vite4中，直接导入会如下报错

Error: Setting up fake worker failed: 
"Cannot read properties of undefined (reading 'WorkerMessageHandler')".

原因

vite2.7 中 pdf.worker.entry.js 除了将pdf.worker.js 转成esmodule后，还将 pdfjsWorker 手动挂载到 window 对象上，而vite4.3中，没有此操作。 但是，在pdfjs的渲染过程中，代码中用到了 window.pdfjsWorker 对象，因此，在 vite 4.3 下通过直接导入 pdf.worker.js 使用时，程序会抛出异常。

2.3 不支持旧版浏览器​

代码在本地调通后发布到线上，发现在钉钉移动端的内置浏览器中打开是空白的，控制台报 SyntaxError 的错误； 最新版本在高版本浏览器中可调通，但低版本不支持，经过查询发现 这个库从 2.4.456 版本开始就不支持旧版本的浏览器了，但钉钉内置浏览器(移动端)属于低版本，所以要降版本。

官网说明，从 2.4.456 版本开始，不再支持旧版浏览器

2.4 无法预览电子签章​

官网在 2.9.359 版本进行了修复，如果是低版本且无法升级的用户，可以通过修改源码进行解决。

注释掉 pdf.worker.js 中的相应代码

2.5 globalthis is not defined​

经排查发现是 globalthis 的兼容性问题，查看 globalThis 兼容性，发现在谷歌71版本和苹果浏览器 12.1 后才支持，问题很明显了，报错的浏览器版本是68，就是globalThis的兼容性问题。

解决方式

  • 1、降版本，版本降至 "pdfjs-dist": "2.0.943"
  • 2、globalthis 在浏览器环境中是指向 window 的，所以在代码中做判断，为空时指向 window 即可。

2.6 第一页缩放有问题​

解决办法：调整比例

function renderPage(num) {
  pdfDoc.getPage(num).then((page)=>{
    const canvasId = `${props.key}-pdf-canvas-${num}`;
    const canvas = document.getElementById(canvasId);
    const ctx = canvas.getContext('2d', { willReadFrequently: true });

    const dpr = window.devicePixelRatio || 1;
    const bsr = ctx.webkitBackingStorePixelRatio ||
                ctx.mozBackingStorePixelRatio ||
                ctx.msBackingStorePixelRatio ||
                ctx.oBackingStorePixelRatio ||
                ctx.backingStorePixelRatio ||
                1
    const ratio = dpr / bsr;
    const viewport = page.getViewport(1, 0, false)
    canvas.width = viewport.width * ratio;
    canvas.height = viewport.height * ratio;
    const aspectRatio = viewport.height/viewport.width;
    const canvasWidth = window.innerWidth||document.body.clientWidth;
    canvas.style.width = canvasWidth + 'px'
    canvas.style.height = canvasWidth * aspectRatio + 'px'
    // ctx.setTransform(ratio, 0, 0, ratio, 0, 0)
    const renderContext = {
      canvasContext: ctx,
      viewport: viewport,
      transform: [ratio, 0, 0, ratio, 0, 0]
    }
    const renderTask = page.render(renderContext);
    renderTask.promise.finally(()=>{
      showLoading.value=false;
    })
    // 在第num页渲染完毕后，递归调用renderPage方法，去渲染下一页，直到所有页面渲染完毕为止
    if (num < pdfPages.value) {
      renderPage(num + 1)
    }
  }).catch(()=>{
    showLoading.value=false;
  })
}

3、正确代码​

<template>
  <div class="pdf-content">
    <canvas
      v-for="pageIndex in pdfPages"
      :id="`${key}-pdf-canvas-${pageIndex}`"
      :key="pageIndex"
    ></canvas>
    <div class="download-content" v-if="isDownload">
      <div
        class="download-content-btn"
        @click="handleDownload"
      >立即下载 <span>（{{ size }}）</span></div>
    </div>
  </div>
</template>
  
<script setup="props" lang="ts">
import { onMounted, ref } from 'vue';
import { useRoute } from 'vue-router';
import { Loading } from 'vant';
import * as Pdfjs from 'pdfjs-dist/build/pdf.js';
import * as pdfjsWorker from 'pdfjs-dist/build/pdf.worker.js';

const props = defineProps<{
  isDownload?: boolean;
  url: string;
  key: string | number;
}>();
const emit = defineEmits(['click', 'download'])

if(!window.pdfjsWorker) {
  window.pdfjsWorker = pdfjsWorker;
}

const route = useRoute();
const {id, size, name} = route.params;

let pdfDoc:any = '';
const pdfPages = ref(0);
onMounted(() => {
  if(props.url) {
    loadFile(props.url)
  }
})

function loadFile(url:string) {
  Pdfjs.GlobalWorkerOptions.workerSrc=pdfjsWorker;
  const loadingTask = Pdfjs.getDocument(url);
  loadingTask.promise.then(pdf=>{
    pdfDoc = pdf;
    pdfPages.value = pdfDoc.numPages;
    // nextTick(()=>{
      renderPage(1)
    // })
  })
}

function renderPage(num) {
  pdfDoc.getPage(num).then((page)=>{
    const canvasId = `${props.key}-pdf-canvas-${num}`;
    const canvas = document.getElementById(canvasId);
    const ctx = canvas.getContext('2d', { willReadFrequently: true });
    const dpr = window.devicePixelRatio || 1;
    const bsr = ctx.webkitBackingStorePixelRatio ||
                ctx.mozBackingStorePixelRatio ||
                ctx.msBackingStorePixelRatio ||
                ctx.oBackingStorePixelRatio ||
                ctx.backingStorePixelRatio ||
                1
    const ratio = dpr / bsr;
    const viewport = page.getViewport(1, 0, false)
    canvas.width = viewport.width * ratio;
    canvas.height = viewport.height * ratio;
    const aspectRatio = viewport.height/viewport.width;
    const canvasWidth = window.innerWidth||document.body.clientWidth;
    canvas.style.width = canvasWidth + 'px'
    canvas.style.height = canvasWidth * aspectRatio + 'px'
    // ctx.setTransform(ratio, 0, 0, ratio, 0, 0)
    const renderContext = {
      canvasContext: ctx,
      viewport: viewport,
      transform: [ratio, 0, 0, ratio, 0, 0]
    }
    page.render(renderContext);
    // 在第num页渲染完毕后，递归调用renderPage方法，去渲染下一页，直到所有页面渲染完毕为止
    if (num < pdfPages.value) {
      renderPage(num + 1)
    }
  }).catch(()=>{
    showLoading.value=false;
  })

}

function handleDownload() {
  emit('download')
}
</script>
